This includes a list of the files and folders with a once sentence purpose description to help a contributor navigate files.
*When modifying these files please* **EDIT** *the qmd files,* **RENDER**, *these as github flavored markdown, and then commit both files to the repository.*

+ README
  - This is the primary landing file for the directory, you are here!
+ 01_MissionVision
  - This states the mission and vision for Soil-DRaH as an organization.
+ 02_Governance
  - Defines decision makers and making bodies (circles) within the project, consent, feedback, and proposal style guides (scope-formulation-decision-revision)
+ 03_PolicyLog
  - Describes the standing policies for the organization including strategic and operational, giving their context, links to the text, metrics, who is responsible, and when it is being revised.
+ 04_CommunityNorms 
  - Community profiles, compassionate communication (NVC) conflict resolution, and abuse reporting.
+ 05_History
  - Project history and major transitions.
+ 06_Goals
  - General project direction and goals including open calls for collaboration and anticipated milestones.
